# Repository readiness

This is the canonical readiness matrix for the selfos repositories. It records
the owner decisions required by [selfos#8](https://github.com/jointsome0-lgtm/selfos/issues/8);
it does not replace each repository's own `AGENTS.md`, issue acceptance gates,
or review rules. Readiness changes only through an explicit owner decision and
a small canonical documentation update. The current cross-system product shape
is the [Integration v1 decision in selfos#25](https://github.com/jointsome0-lgtm/selfos/issues/25).
