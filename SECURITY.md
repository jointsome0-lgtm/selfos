# Security Policy

This policy covers the selfos ecosystem repositories under
[github.com/jointsome0-lgtm](https://github.com/jointsome0-lgtm):
**selfos** (this repo), **atlas**, **exp2res**, **ephemeris**, and
**selfos-skills**. Subsystem repositories link here instead of restating it.

## Reporting a vulnerability

- Preferred: GitHub private vulnerability reporting — "Report a
  vulnerability" under the Security tab of the affected repository.
- Alternative: email <jointsome0@gmail.com>.
- Please do not open a public issue for a vulnerability before it is fixed.

The portfolio is solo-maintained; response is best effort, with an
acknowledgment normally within a week.

## Data exposure counts as a vulnerability

Public repositories in this ecosystem must never contain real personal data —
only invented fixtures authored by the synthetic persona (see
[docs/architecture.md](docs/architecture.md) and
[docs/hygiene.md](docs/hygiene.md)). If you find anything in a public
repository that looks like real diary text, health data, private journals, or
other personal records, report it through the channels above exactly as you
would a code vulnerability.
